🔒 Kebijakan Privasi

Tanggal Berlaku: 23 April 2026

1. Prinsip Tanpa Penyimpanan Data

AstroScan beroperasi dengan kebijakan tanpa penyimpanan data yang ketat di server kami. Kami tidak mengumpulkan, menyimpan, atau mencatat informasi pribadi apa pun yang dimasukkan ke dalam tes atau bot Telegram di server mana pun. Data kelahiran diproses secara sementara dan dibuang secara permanen setelah laporan dibuat.

Situs web tes itu sendiri hanya menyimpan dua preferensi kecil hanya di penyimpanan lokal browser Anda sendiri (bahasa yang Anda pilih, dan nilai pelacak suasana hati opsional) — data ini tidak pernah meninggalkan perangkat Anda, tidak pernah dikirimkan kepada kami atau siapa pun, dan sepenuhnya berada di bawah kendali Anda (menghapus data browser Anda menghapusnya secara instan).

2. Integrasi Telegram

Semua layanan tambahan disediakan melalui @ASTROSCAN_BOT. Pembayaran diproses melalui Telegram Stars; kami tidak memiliki akses ke detail pembayaran Anda. Lihat Kebijakan Privasi Telegram untuk lebih lanjut.

3. Analitik

Situs web ini menggunakan analitik yang ramah privasi (LibreCounter, OpenCounterAPI) yang tidak menggunakan cookie dan tidak melacak individu. Hanya jumlah kunjungan agregat dan anonim yang disimpan.

4. Tanpa Berbagi dengan Pihak Ketiga

Kami tidak menjual, memperdagangkan, atau mentransfer data pengguna apa pun. Karena kami tidak menyimpan data, tidak ada yang perlu dibagikan.

Berdasarkan UU Perlindungan Data Pribadi (UU PDP), Anda memiliki hak untuk mengakses, memperbaiki, dan meminta penghapusan data pribadi Anda — namun karena kami tidak menyimpan data apa pun di server kami, hak-hak ini tidak memiliki data untuk diterapkan.

5. Perubahan

Pembaruan akan diposting di halaman ini dengan tanggal berlaku yang direvisi.
